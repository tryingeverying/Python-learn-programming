10. - hex(x) 将整数x转换为以'0x'为前缀的十六进制字符串
    - oct(x) 将整数x转换为以'0o'为前缀的八进制字符串
    - bin(x) 将整数x转换为以'0b'为前缀的二进制字符串
    - ord(x) 将字符x转化为对应ASCII码表中的序号
    - chr(x) 将整数x转化为对应ASCII码表中的字符
    - 上述的整数是任何进制都可以
